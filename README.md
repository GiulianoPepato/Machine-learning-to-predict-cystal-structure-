# Machine-learning-to-predict-cystal-structure-based-on-atomic-features
O presente reposório contém toda a parte técnica (códigos e arquivos html) gerados para realizar a minha iniciação científica orientado pelo professor Gustavo Dalpian e financiada pela CNPq. O tema do projeto é "Usar Machine Learning para predizer a Estrutura Cristalina de Materiais baseado em Features Atômicas", que é motivado pelo fato de que processos de sintetização, caraceterização, cálculos de primeiros principios e de DFT para estimar essa propriedade requerem um alto consumo de recursos materiais, computacionais e financeiros, muito diferente de uma abordagem de ML, que custa muito menos e é muito mais rápida embora possua uma precisão menor. Por esse motivo, é  importante estudar que conjuntos de features baseadas apenas nos elementos químicos que compõem o material e qual algoritmo de Machine Learning que melhor otimiza essa previsão e comparar com demais trabalhados da literatura. Todo o código feito para avaliar esse processo pode ser acessado no arquivo `ML_crystal_structure.ipynb`. Caso queria análisar o artigo, me contate via giuliano.pepato@usp.br.
